## 类型选择说明

- `str`：保存姓名、刺激词、墨水颜色和反应键。
- `float`：保存反应时和平均分。
- `bool`：保存是否通过、是否正确。
- `list`：保存多次练习分数和技能清单。
- `dict`：保存一名学生或一次 trial 的完整记录。
- `None`：保存暂时还没有安排的下次复习时间。
